# HoyaDB
HoyaDB is a simple in-memory key-value store database written in Rust.

## Curently Implemented
- [x] REPL
- [x] Basic implementation of a Database
- [x] Basic parser
- [x] Basic typechecker
- [x] Multithreading (kinda)
